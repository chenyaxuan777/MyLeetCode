二叉树的结构：  
```Python
class TreeNode:
    def __init__(self, x):
        self.val = x
        self.left = None
        self.right = None
```

**后序遍历**
递归写法：
···Python
class Solution:
    def postorderTraversal(self, root: TreeNode) -> [int]:
        if root == None:
            return []
        else:
            return self.postorderTraversal(root.left) + self.postorderTraversal(root.right) + [root.val]
```
